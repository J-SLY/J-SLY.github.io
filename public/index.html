如何投稿？

两种方式任选其一：

  1. GitHub — 点击上方「GitHub 投稿」按钮，按模板填写 Issue 提交（你的 GitHub 用户名会自动作为作者名）。
  2. 邮件 — 点击上方「邮件投稿」按钮，发送邮件至 submit@support.jsly.asia 或 2029872426@qq.com，请附上文章标题、正文（Markdown）和你的署名。
  3. 提交后等待审核，审核通过后文章会出现在这里。
